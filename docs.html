+ Birdata API Intro

Birdata is an effort to compile a list of all birds native to the eastern
US. We like to keep the site open to the public. Part
of this is allowing our search tools and database to be accessible to anyone..



+ How to add a Birdata search bar to your page

Just add to the body of your site the below code:
<input id="search"></input> <br><input id="name" type="checkbox" checked> <label for="name">Search by name or by family</label> <br><button onclick="searchf();">Search</button> <br><p id="result"></p><script>function searchf(){ let r; let t="family"; if (document.getElementById("name").checked){ t="name"; } r=search({"keyword":document.getElementById("search").value,"type":t, "order":true}) t=[]; for (var i=0; i < r.length; i ++){ t.push("<h3>Name: "+r[i].name+"</h3><br><p>Family: "+r[i].type[0]+"s/"+r[i].type[1]+"</p><br><hr>") } document.getElementById("result").innerHTML=t; }</script>



+ Adjusting Searchbar Settings

If you want, you can adjust the JSON in the bit that says "Search" to make it sort differently.
